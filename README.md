# HACKIE: THE AI CORE 💠

**The ultimate Bluetooth HID injection & AI synthesis core for professional environments.**

Hackie (formerly Rabit) is a high-performance Android tool that transforms your mobile device into a stealthy, high-speed Bluetooth HID (Human Interface Device) injector. With integrated Gemini 1.5 Pro AI and a local encrypted vault, Hackie bridges the gap between hardware input and machine intelligence.

### 🧠 AI SYNAPSE
Real-time AI logic via Gemini 1.5 Pro. Generate code, shell scripts, or documentation and inject them directly into the target system's active cursor.

AES-256 encrypted local storage for credentials and sensitive macros. Protected by hardware-backed Biometric (Fingerprint/Face) handshakes.

Zero-latency multi-touch touchpad translation. Transform your screen into a precision HID cursor driver.

## 🚀 QUICK START

### 1. INITIALIZATION
Download the latest [Hackie APK](https://github.com/Sagarverse/Hackie/releases). Ensure Bluetooth and location (for BLE scanning) permissions are granted.

### 2. ESTABLISH LINK
- Enable Bluetooth on the target machine (Mac/PC).
- Open Hackie and select **Pairing Mode**.
- Connect via the system Bluetooth settings (it will appear as a "Standard Keyboard/Mouse").

### 3. PROTOCOL_EXECUTE
Select a module (Control, Synapse, or Vault) and begin the injection flow.
