COMMONLY ASKED

Is Windows supported?

Currently, Hackie is optimized for macOS and Android targets. Windows support is in the experimental roadmap for v2.7.

Does it require an internet connection?

Standard HID features work offline. AI Synapse features require an active connection to process LLM requests via the API.

Is my Password Vault secure?

Yes. We use Android's EncryptedSharedPreferences (Hardware-backed keystore). No data is ever synced to the cloud.
